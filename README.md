
# SmartStockSuite

SmartStockSuite is a smart inventory management system created as a final-year Software Engineering project.

The system helps manage warehouse stock, products, suppliers, purchase orders, customer orders, product locations, picking tasks, stock movements and audit logs.

## Main Features

- Product management
- Supplier management
- Purchase order management
- Customer order management
- Product location management
- Picking task management
- Barcode / QR scanning support
- Stock movement tracking
- Audit logging
- Role-based access control
- Web dashboard
- Mobile scanner concept

## Technologies Used

- ASP.NET Core Web API
- C#
- PostgreSQL
- Docker
- React / TypeScript
- Android Studio
- Kotlin
- GitHub

## User Roles

### Manager
The manager has full system access and can manage products, suppliers, users, locations, orders and reports.

### Admin
The admin can monitor stock, view reports, check tasks and support warehouse operations.

### Picker / User
The picker can use the mobile application to view assigned picking tasks, scan products and complete picking work.

## Project Purpose

The purpose of this project was to design and develop a practical warehouse inventory system that improves stock control, order picking accuracy and traceability.
